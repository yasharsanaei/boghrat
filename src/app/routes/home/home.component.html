View excel files data!

In this app you will upload a CSV file and it will show the contents of the file in a table with this rules:

  • The first and last name must have at least three letters.
  • The mobile number pattern must be checked.
  • The national code pattern, which can be searched on the Internet, must be checked.
  • The age should be between 1 and 100.

Also, the average age of all people on the current page will be displayed in a box above the table.
